# recategorizacion-calculadora-upcn
Calculadora orientativa, donde podes cargar tus datos y conocer qué categoría podría corresponderte, según los criterios generales del Instructivo de Recategorización. El resultado es meramente orientativo, no es definitivo ni vinculante, y siempre está sujeto a la evaluación de la autoridad competente y a las particularidades de cada situación.
